## Installation and Setup
1-) Clone the repo
```
git clone https://github.com/P-pisiko/Native-library.git
```
2-) Navigate in to the project directory
```
cd  Native-library
```
3-) Install dependencies 
```
npm install
```

## When the dependencies install you can start the application.
```
npm start run
```
